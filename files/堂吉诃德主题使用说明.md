# 堂吉诃德艺术风格主题使用说明

## 🎨 主题简介

本主题受堂吉诃德经典黑白剪影插画启发,为博客网站带来独特的文学艺术气质。

**主要特点:**
- 🎭 黑白剪影色调
- 📜 羊皮纸纹理背景
- ✒️ 古典衬线字体
- 🖋️ 手写风格标题
- 📚 文学装饰元素

## 🔧 启用主题

### 方法1: 配置文件开关(推荐)

在 `_config.yml` 中设置:

```yaml
don_quixote_style: true  # 启用堂吉诃德风格
```

设置为 `false` 可关闭主题,恢复默认样式。

### 方法2: 完全移除

如需完全移除主题,删除以下文件:
- `_sass/_don-quixote-theme.scss`
- `assets/css/quixote-fonts.css`

并在 `assets/css/main.scss` 中删除导入语句:
```scss
"don-quixote-theme"
```

## 🎨 主题配色方案

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主色调 | `#1a1a1a` | 页头、边框、文字 |
| 背景色 | `#fefef8` | 页面背景(米黄色) |
| 强调色 | `#d4a574` | 装饰、悬停效果 |
| 文字色 | `#2d2d2d` | 正文内容 |
| 边框色 | `#3d3d3d` | 卡片、分隔线 |

## 📝 字体配置

主题使用 Google Fonts 提供的古典字体:

- **Cinzel**: 优雅标题字体
- **EB Garamond**: 经典正文字体  
- **Crimson Text**: 阅读友好字体
- **Philosopher**: 手写风格字体

这些字体通过 `assets/css/quixote-fonts.css` 自动加载。

## 🔄 自定义调整

### 修改配色

编辑 `_sass/_don-quixote-theme.scss` 文件顶部的变量:

```scss
$quixote-primary: #1a1a1a;   // 主色调
$quixote-bg: #fefef8;         // 背景色
$quixote-accent: #d4a574;     // 强调色
```

### 调整字体

编辑 `assets/css/quixote-fonts.css`:

```css
:root {
  --font-display: 'Cinzel', Georgia, serif;
  --font-heading: 'EB Garamond', Georgia, serif;
  --font-body: 'Crimson Text', Georgia, serif;
}
```

### 关闭动画

在 `_don-quixote-theme.scss` 末尾删除或注释:

```scss
/* 
.archive__item,
.page__content {
  animation: ink-drop 0.6s ease-out;
}
*/
```

## 📱 响应式设计

主题自动继承原网站的响应式布局,在移动端、平板和桌面设备上均能良好显示。

## 🐛 已知问题

1. **字体加载**: Google Fonts 可能在国内访问较慢,可考虑使用本地字体
2. **浏览器兼容**: 部分CSS特性需要现代浏览器支持
3. **颜色对比**: 黑白配色适合阅读,但装饰元素可能不够醒目

## 💡 最佳实践

1. **内容为王**: 主题强调文字内容,适合长篇技术文章和散文
2. **配图建议**: 使用黑白或复古风格图片更搭配主题
3. **代码高亮**: 建议使用深色主题的代码高亮方案
