## 🔧 如何使用

### 启用主题
在 `_config.yml` 中:
```yaml
don_quixote_style: true
```

### 关闭主题
改为:
```yaml
don_quixote_style: false
```

### 自定义颜色
编辑 `_sass/_don-quixote-theme.scss`:
```scss
$quixote-primary: #1a1a1a;  // 主色调
$quixote-bg: #fefef8;        // 背景色  
$quixote-accent: #d4a574;    // 强调色
```
