## 如何使用

### 发布新文章

1. 在 `_posts` 目录下创建新文件
2. 文件命名格式: `YYYY-MM-DD-文章名.md`
3. 文件开头添加 Front Matter:

```yaml
---
title: '文章标题'
date: 2025-11-04
permalink: /posts/2025/11/article-url/
tags:
  - 标签1
  - 标签2
categories:
  - 分类名
---

在这里写文章内容...
```

### 本地预览

```bash
# 安装依赖
bundle install

# 启动本地服务器
bundle exec jekyll serve

# 浏览器访问 http://localhost:4000
```

### 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 master 分支作为源
4. 网站将自动部署

## 自定义建议

### 修改个人信息
编辑 `_config.yml` 中的 `author` 部分

### 更换头像
将图片放在 `images/` 目录,然后在 `_config.yml` 中修改 `avatar` 字段

### 添加评论功能
在 `_config.yml` 中配置评论系统 (如 Disqus):

```yaml
comments:
  provider: "disqus"
  disqus:
    shortname: "your-disqus-shortname"
```

### 修改主题颜色
编辑 `_sass/_themes.scss` 文件
